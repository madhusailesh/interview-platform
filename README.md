# SaileshHire 🚀

**SaileshHire** is a modern real-time technical interview platform built using the MERN stack.  
It allows interviewers and candidates to communicate seamlessly through video meetings, collaborative coding, chat, and interview scheduling — all in one place.

---

# ✨ Features

- 🎥 **Real-time Video Calling** — High-quality peer-to-peer video connection.
- 💻 **Collaborative Code Editor** — Real-time code sharing during interviews.
- 📅 **Interview Scheduling System** — Book and manage interviews seamlessly.
- 🔐 **JWT Authentication & Authorization** — Secure login and role-based access control.
- 👨‍💼 **Separate Dashboards** — Dedicated views for Interviewers & Candidates.
- ⚡ **Real-time Communication** — Powered by Socket.IO for instant updates.
- 🖥️ **Monaco Editor Integration** — VS Code-like coding experience.
- 📱 **Fully Responsive UI** — Works smoothly on all screen sizes.
- 🌙 **Modern Dark UI** — Styled beautifully with Tailwind CSS.

# ⚙️ Installation

## 1️⃣ Clone the Repository

```bash
git clone https://github.com/madhusailesh/interview-platform.git
```


## 3️⃣ Install Dependencies

### For Client (Frontend)

```bash
cd client
npm install
```

### For Server (Backend)

```bash
cd ../server
npm install
```

---

# 🔑 Environment Variables

Create a `.env` file inside the `server` folder and add:

```env
PORT=5000

MONGO_URI=your_mongodb_connection_string

JWT_SECRET=your_secret_key
```

---

# ▶️ Run the Application

## Start Backend

```bash
cd server
npm start
```

## Start Frontend

```bash
cd client
npm run dev
```
